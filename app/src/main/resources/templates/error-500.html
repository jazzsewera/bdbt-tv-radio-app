500 Internal Server Error

What happened?

The server encountered an unexpected situation that it could not handle.

Why did it happen?

Probably because you violated the database constraints, e.g. tried to add an entity with an existing id, or set the foreign key to a nonexistent one.

What can I do?

You can go back to the Homepage, People, Productions, or Tasks, and try again. If it does not help, it means there is a problem with this site.

I want to know more

There are a lot of good resources where you can learn more about HTTP response codes. Our favorite is Mozilla's web doc on 500 Status Code.
